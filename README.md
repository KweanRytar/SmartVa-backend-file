SmartVA Backend

SmartVA Backend is a scalable Node.js and Express.js API that powers SmartVA, a productivity platform for virtual and onsite administrative assistants.
It handles authentication, event scheduling, reminders, notifications, and secure user-specific data using MongoDB.
